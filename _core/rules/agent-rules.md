# agent-rules.md

이 워크스페이스에서 AI 에이전트가 따르는 행동 규칙을 정의한다.

## 1. 허용 행동

다음 행동은 사용자 승인 없이 수행할 수 있다.

- `_core/` 내 파일 읽기·수정·생성
- `shared/` 내 파일 읽기·수정·생성
- `templates/` 내 파일 읽기·수정·생성
- `guides/` 내 파일 읽기·수정·생성
- `scripts/` 스크립트 실행 (dry-run에 한함)

## 2. 금지 행동

다음 행동은 어떠한 경우에도 수행하지 않는다.

- 사용자 승인 없이 `templates/` 하위 파일 삭제
- 루트 파일(`README.md`, `CLAUDE.md`, `AGENTS.md`) 무단 수정
- `_core/refs/` 파일 수정 (참조 전용)
- `scripts/` 스크립트 `--apply` 옵션 무단 실행

## 3. 승인 절차

영향이 큰 작업은 반드시 실행 전에 사용자 승인을 받는다.

| 작업 | 승인 방식 |
|---|---|
| 템플릿 루트 구조 변경 | 변경 목록 제시 후 확인 |
| `shared/` → `templates/` 동기화 (`--apply`) | dry-run 결과 보고 후 실행 |
| 파일·폴더 삭제 | 삭제 대상 명시 후 확인 |

## 4. 작업 후 필수 갱신

`templates/` 구조를 변경한 경우 아래 파일을 즉시 갱신한다.

- `guides/` 내 해당 템플릿 설명 문서
