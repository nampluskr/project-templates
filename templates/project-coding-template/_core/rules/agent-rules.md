# agent-rules.md

이 프로젝트에서 AI 에이전트가 따르는 행동 규칙을 정의한다.

## 1. 허용 행동

- 프로젝트 루트 내 모든 파일 읽기·수정·생성
- `_core/sessions/` 에 핸드오프 문서 생성
- `_core/docs/project-log.md` 갱신

## 2. 금지 행동

- 사용자 승인 없이 루트 폴더 구조 변경
- `_core/rules/` 파일 수정 (규칙은 사용자가 관리)
- 프로젝트 외부 경로 접근

## 3. 승인 절차

영향이 큰 작업은 반드시 실행 전에 사용자 승인을 받는다.

| 작업 | 승인 방식 |
|---|---|
| 루트 폴더 구조 추가·삭제 | 변경 목록 제시 후 확인 |
| 파일 일괄 삭제 | 삭제 대상 명시 후 확인 |

## 4. 규칙 적용 기준

작업 유형에 따라 적용할 규칙 파일이 다르다.

| 작업 유형 | 적용 규칙 |
|---|---|
| 마크다운 문서 작성·수정 | `_core/rules/docs-rules.md` |
| Python 스크립트 작성·수정 | `_core/rules/python-rules.md` |

## 5. 응답 원칙

- 한국어 경어체로 응답한다.
- 변경한 파일과 결과를 간단히 요약한다.
- 영향이 큰 작업은 실행 전에 의도를 먼저 설명한다.
